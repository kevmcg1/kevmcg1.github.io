Guide

Welcome

This tool draws a 3D picture of a CNC program (GCode) so you can see exactly where the cutting tool will move before you run it on a real machine.

Quick Start

  • Click a Sample card on the left to load a demo program.
  • Or paste your own GCode into the editor at the bottom.
  • Press Play (▶) at the bottom of the viewport to watch the tool move.

Moving the View

  • Left drag — slide the view around (pan).
  • Right drag — orbit/rotate around the part.
  • Scroll wheel — zoom in and out.
  • Click the little cube in the top-right to snap to a side view (Top, Front, Right, etc.).

What the Colors Mean

  • Red line — X axis
  • Green line — Y axis
  • Blue line — Z axis (up/down)
💡 Tip: When you click Right, Left, Front, or Back view, the arrowheads on the axes hide so the view stays clean. They come back as soon as you orbit again.

Supported GCode

  • Motion: G0, G1, G2, G3 (linear & arcs with I/J or R)
  • Modes: G90/G91 (abs/rel), G20/G21 (in/mm)
  • Canned cycles: G81 (drill), G82 (dwell), G83 (peck), G84 (tap)
  • Return: G98/G99 (initial Z / R-plane)
  • Stops: M00 (mandatory), M01 (optional)
  • Coolant: M07/M08/M09

Tool Table

Each T word in the program shows up in the Tool Table on the left. Edit the diameter (in or mm) to change how big the simulated tool cylinder appears.

ISO Follow Mode

Locks the camera to a fixed isometric angle while tracking the tool tip. Turn on Rotate with toolpath to also have the camera spin to face the direction of travel — useful for inspecting long programs from the tool's perspective.

Playback Controls

  • Single Block — pauses after every program line so you can step through one move at a time.
  • Speed slider — ranges from 0.25× to 10×.
  • Progress slider — scrub through the program by percentage.

Display Toggles

  • Focus dim on cursor line — fades everything except the move on the line where the editor cursor sits.
  • Show full toolpath ahead — draws future moves at 20% opacity during playback.
  • Show arc centers — marks the I/J center point of every G2/G3 arc.
⚙ Coordinate convention: Z+ is up, X+ is right, Y+ is into the screen (right-hand rule).
Settings

The Basics

The most-used settings live on the left sidebar. Here are the ones that change the look of the visualizer the most:

  • Show Grid — turn the floor grid on or off.
  • Show Axes — turn the X/Y/Z reference lines on or off.
  • Show Axis Letters — show or hide the X/Y/Z labels at the end of each axis.
  • Show tool cylinder — show or hide the simulated tool while it cuts.

Grid Size

Under Grid Settings you can change how big the grid is and how big each square is. Pick inch or mm depending on which units your program uses.

💡 If your part looks tiny or huge, hit Fit to View at the bottom of the sidebar — it auto-zooms so the whole toolpath fits on screen.

Display Tuning

  • Focus dim on cursor line — when ON, all moves except the one matching the editor cursor's line are dimmed. Great for tracing a specific block.
  • Show arc centers — overlays the center point and radius of every G2/G3. Helpful for debugging arc errors.
  • Show full toolpath ahead — during playback, future moves render at 20% alpha so you can see what's coming.
  • Optional stop (M01) — when ON, M01 commands pause the simulation. Toggle OFF to skip them.

Visibility Toggles

The Visibility section controls scene chrome (grid + axes + axis letters). All can be combined for a minimal "just the toolpath" look.

Tool Diameter Units

Inside Tool Table you can switch the diameter input between in and mm. Values are converted on the fly — the underlying tool size doesn't change, just the unit you type in.

Camera Behavior

  • Camera Views snap instantly. Right, Left, Front, Back hide the axis arrowheads so axes look like clean reference lines. Orbit again to bring them back.
  • ISO Follow → Rotate with toolpath locks the orbit, but pan still works unless follow mode is on.
  • Reset Camera returns to the default ISO angle and 1.0× zoom.
⚙ All settings are session-only — they reset on page reload.
